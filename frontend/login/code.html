Document-first workspace

문서 작업을 시작할 사용자를 선택하세요.

MVP1에서는 아직 실제 인증을 사용하지 않습니다. 이 화면에서 생성하거나 조회한 사용자 ID를 현재 사용자로 저장하고, 이후 프로젝트와 문서 API 호출에 사용합니다.

1

사용자 생성 또는 선택

이름과 이메일로 새 사용자를 만들거나 기존 사용자 ID를 조회합니다.

2

현재 사용자 저장

선택된 userId는 브라우저 상태에 저장되어 다음 화면에서 재사용됩니다.

3

프로젝트 작업으로 이동

저장된 사용자는 프로젝트 생성의 ownerUserId, 조회의 requesterUserId로 쓰입니다.

시작하기

현재 백엔드 API 기준으로 사용자 생성과 기존 사용자 조회를 지원합니다.
